# Serverpod Cloud Clients

This repository contains the client libraries for the Serverpod Cloud. The client libraries are used to communicate with the Serverpod Cloud API. 


The Serverpod Cloud Clients are developed in a closed-source repository and then published here. We are not accepting contributions to this repository at this time. 

Please create an issue in the main [Serverpod repository](https://github.com/serverpod/serverpod/issues) or on the feedback board in the [console](https://console.serverpod.cloud/feedback) if you have any suggestions or feedback.

For support related requests or reach out to us on [Discord](https://discord.com/invite/wJ4pQeHhVc) or send an email to [support@serverpod.cloud](mailto:support@serverpod.cloud).

You can find the general cloud documentation [here](https://docs.serverpod.cloud).
